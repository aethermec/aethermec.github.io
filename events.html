Brain Wars

BrainWars is a two-round event that tests a participant’s ability to solve logical problems. Brain- Wars consists of two rounds. The first round is an elimination round where 40% of the teams can qualify for the second round which shall be a visual round.

Valid under Aether Pass
  • Prize Money:
    • INR 6,000
  • Duration: 3 hours
  • Rules and Regulations:
    • 4 surprise rounds
    • Cross talk between teams is prohibited once the event starts.
    • Usage of electronic devices (laptops, phones, smart watches, calculators) is prohibited.
  • Event Structure:
    1. Round One:
      1. The first round is a pen and paper round in which the participants will get a set of fifteen questions that will test logic and common sense.
      2. The marking scheme for each question will be mentioned on the question paper.
      3. Participants are not allowed to leave the room before submitting their paper to the invigilator.
      4. Pens will be provided.
    2. Round Two:
      1. The second round is a visual round where you’ll be asked questions based on pictures and videos shown by the projector.
      2. Each team will be handed a blank paper in which they have to write the answers to the questions that shall be displayed on the screen.
  • Team Size: 1 - 3
    Note: The invigilating team has the authority to disqualify the participants of the whole team in case of violation of the above mentioned rules.

Reverse Coding

Reverse coding is a fun event in which the participants will be given some codes which they can execute but the source code isn't visible. They can run the code multiple times giving an input of their choice and observing the output. Looking at the input output patterns they need to figure out what is going on beneath the hood and code it.

Valid under Aether Pass
  • Prize Money:
    1. INR 5000 +Merchandise
  • Duration: ** Hours
  • Rules and Regulations:
    1. Each team will be given 4 questions to solve.
    2. Questions will have points based on difficulty.
    3. For a given question the participant is allowed to run the given code multiple no of times. But will not have access to the code.
    4. Based on the output he receives for given values he enters, he has to decipher how the code function and is required to reproduce it.
    5. The code he writes is run against a set of test cases and the number of test cases passed determines the no of points he gets.
    6. The participants are allowed to code in python , c ,c++ or java.
    7. Pseudo code’s will be accepted but fraction of the points will be allotted.
    8. The event will have only one round.
    9. The participants will have no internet access
  • Team Size: 1 - 2
